NuggetBotSqueak Official Website

Welcome to the official website for the NuggetBotSqueak Discord bot! Here you can find links to the development server, support server, and more. If you don't know what this bot does, keep reading!

About NuggetBotSqueak

NuggetBotSqueak is a general multipurpose Discord bot that is written in Node.js and uses Discord.js (you probably don't care, so I don't know why this is here). It has yes/no polls, advanced polls, an arson command, swear censoring, and more! NuggetBotSqueak's prefix is ;. (If you don't know what that means - highly unlikely - it means that to run the help command, you would send the message ;help).

Features (in detail)

This is the command format used: [whatever] is a required one-word option. [two words]represents a multiple word option. This means that there will be seperators between options. {something} represents an optional single-word option. {multiple word optionrepresents an optional multiple word option. Multiple-word options will always have a certain seperator (i.e. ;;) between each other. [#channel] means you have to mention a channel, and [@user] requires a user mention.

Censoring

When a user sends a message that includes anything from the blacklist, NuggetBotSqueak will delete the message and send a log message in the server's system channel in the following format: "Deleted message from ${user} in ${channel}. Timestamp: ${MM}/${dd}/${yyyy} ${HH}:${mm}:${ss}". A complete list of swears can be found in the code. The blacklist is not made readily available for a reason.

Polls

Yes/No Polls

The simpoll command creates an embedded yes/no poll. The command's format is as follows: ;simpoll [#poll-channel] [poll question].

Advanced Polls

The advpoll command creates an embedded advanced poll. The command's format is as follows: ;advpoll [#poll-channel] [poll question] ;q; [option 1] ;; [option 2] ;; {option 3} ;; {option 4} ;; {option 5} ;; {option 6} ;; {option 7} ;; {option 8} ;; {option 9} ;; {option 10}.

Announcements

The announce command creates an embedded announcement in the specified channel. The command format is as follows: ;announce [#announcement-channel] [announcement message].

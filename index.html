About NuggetBotSqueak

NuggetBotSqueak is a general multipurpose Discord bot that is written in Node.js and uses Discord.js (you probably don't care, so I don't know why this is here). It has yes/no polls, advanced polls, an arson command, swear censoring, and more! NuggetBotSqueak's prefix is ;. (If you don't know what that means - highly unlikely - it means that to run the help command, you would send the message ;help).

Features (in detail)

Censoring

When a user sends a message that includes anything from the blacklist, NuggetBotSqueak will delete the message and send a log message in the server's system channel in the following format: "Deleted message from ${user} in ${channel}. Timestamp: ${MM}/${dd}/${yyyy} ${HH}:${mm}:${ss}". A complete list of swears can be found in the code. The blacklist is not made readily available for a reason.
